h2. Ψευδώνυμα ηλεκτρονικού ταχυδρομείου

Ένα ψευδώνυμο (alias) είναι μια διαφορετική διεύθυνση ηλεκτρονικού ταχυδρομείου που σε διάφορες καταστάσεις μπορεί να χρησιμοποιηθεί για την αποστολή και λήψη email χωρίς τη δημιουργία νέου λογαριασμού Riseup.

Για παράδειγμα, εάν πρέπει να χρησιμοποιήσετε μια διεύθυνση email κατά την εγγραφή σας σε έναν ιστότοπο, αντί να χρησιμοποιείτε την κανονική σας διεύθυνση email, μπορείτε να δημιουργήσετε ένα ψευδώνυμο και να το χρησιμοποιήσετε. Με αυτόν τον τρόπο θα γνωρίζετε εάν αυτός ο ιστότοπος αρχίσει να σας στέλνει ανεπιθύμητα μηνύματα, και αν ποτέ θελήσετε να σταματήσετε να λαμβάνετε μηνύματα ηλεκτρονικού ταχυδρομείου από αυτούς, μπορείτε να διαγράψετε αυτό το ψευδώνυμο.

h3. Λήψη και αποστολή email με riseup ψευδώνυμο 

Εάν προσθέσετε ένα riseup.net ψευδώνυμο στον λογαριασμό σας, τα μηνύματα που αποστέλλονται σε αυτήν τη διεύθυνση θα παραδίδονται στο mailbox του λογαριασμού σας.

Για παράδειγμα, ο λογαριασμός σας μπορεί να είναι joe_hill@riseup.net, αλλά να θέλετε να χρησιμοποιήσετε τη διεύθυνση maydaychicago@riseup.net ως διεύθυνση επικοινωνίας για μια ενέργεια. Τα email που εστάλησαν στο ψευδώνυμο maydaychicago@riseup.net θα παραδοθούν στο joe_hill@riseup.net mailbox.

Μπορείτε επίσης να στείλετε μηνύματα χρησιμοποιώντας αυτό το ψευδώνυμο ως τη "From" διεύθυνση.

h3. Πώς μπορώ να δημιουργήσω ένα νέο ψευδώνυμο;

Για να δημιουργήσετε ένα νέο ψευδώνυμο:

# Συνδεθείτε στο https://account.riseup.net
# Πατήστε στο "Mail Settings" στα αριστερά
# Πατήστε στην καρτέλα "Aliases" πάνω πάνω.
# Πληκτρολογήστε το ψευδώνυμο σας (πχ alias@riseup.net) στο πεδίο με την ένδειξη "New Alias".
# Κάντε κλικ στο πράσινο κουμπί "Create".

Εάν το ψευδώνυμο που επιλέξατε είναι διαθέσιμο τότε θα εμφανιστεί στη λίστα σας με ετικέτες "Send" και "Receive"..

h3. Πώς μπορώ να διαγράψω ένα ψευδώνυμο;

Για να αφαιρέσετε ένα ψευδώνυμο κάντε κλικ στο κουμπί του κάδου απορριμμάτων στα δεξιά του ψευδωνύμου. Θα λάβετε ένα μήνυμα που θα λέει "changes saved" και το ψευδώνυμο θα εξαφανιστεί από τη λίστα σας.

h3. Πώς μπορώ να διαμορφώσω τον email client μου ώστε να στέλνει μηνύματα ηλεκτρονικού ταχυδρομείου χρησιμοποιώντας το ψευδώνυμό μου;

Αυτό θα εξαρτηθεί από τον email client που χρησιμοποιείτε. Εάν χρησιμοποιείτε το webmail του Riseup, ακολουθήστε τα παρακάτω βήματα:

# Συνδεθείτε στο https://mail.riseup.net
# Πατήστε στο "Settings" πάνω δεξιά
# Πατήστε στο "Identities" στα αριστερά
# Πατήστε το "+" για να προσθέσετε ένα ψευδώνυμο
# Συμπληρώστε τα πεδία διασφαλίζοντας ότι θα προσθέσετε το ψευδώνυμο τουλάχιστον στο πεδίο "Email"
# Στείλτε emails με το ψευδώνυμό σας
## Πατήστε στο "Send" πάνω αριστερά
## Επιλέξτε το ψευδώνυμό σας από την αναπτυσσόμενη λίστα με την ετικέτα "From"
## Συμπληρώστε το email

Εάν χρησιμοποιείτε άλλον client, όπως το Thunderbird τότε θα χρειαστεί να ορίσετε την κεφαλίδα "From" στο ψευδώνυμό σας. Οι ρυθμίσεις SMTP θα πρέπει να χρησιμοποιούν την κανονική σας διεύθυνση email για έλεγχο ταυτότητας.

h3. Προηγμένη χρήση

Το Riseup επιτρέπει επίσης την καταχώρηση ψευδωνύμων για 3rd party domains. **Δεν σας συνιστούμε να δοκιμάσετε την προηγμένη χρήση** εάν δεν είστε εξοικειωμένοι με την προηγμένη χρήση των email. Εάν ρυθμίσετε εσφαλμένα το 3rd party ψευδώνυμο, **είναι πιθανό να χάσετε τα μηνύματα που σας αποστέλλονται**.

Όταν προσθέτετε ένα νέο 3rd party ψευδώνυμο, θα πρέπει να επιβεβαιώσετε ότι έχετε πρόσβαση στο email κάνοντας κλικ σε έναν σύνδεσμο επιβεβαίωσης που σας έχει αποσταλεί από το Riseup. Εαν δεν λάβετε το email, ελέγξτε τον Spam/Junk φάκελο η πατήστε στο κουμπί "Resend Confirmation Email" από την λίστα ψευδωνύμων.

h4. Λήψη email με 3rd party ψευδώνυμο

Εάν προσθέσετε ένα ψευδώνυμο για ένα domain διαφορετικό από του riseup.net, τα emails που αποστέλλονται σε αυτήν τη διεύθυνση δεν θα φτάνουν αυτόματα στον riseup λογαριασμό σας. Σας επιτρέπεται να λαμβάνετε μηνύματα ηλεκτρονικού ταχυδρομείου εάν ρυθμίσετε ένα email για προώθηση στη διεύθυνσή σας στο riseup.net.

Σημειώστε ότι τα μηνύματα που προέρχονται από 3rd party ψευδώνυμα θα περνάν πάντα πρώτα από αυτούς τους 3rd party διακομιστές και θα υπόκεινται στις πολιτικές φιλτραρίσματος τους καθώς και στις επιτηρήσεις τους.

Σημειώστε επίσης ότι ακόμα κι αν ρυθμίσετε σωστά αυτήν τη δυνατότητα, η ετικέτα "Receive" στη λίστα ψευδωνύμων δεν θα εμφανίζεται για 3rd party ψευδώνυμα. Αυτό προορίζεται για riseup.net ψευδώνυμα, δεδομένου ότι οι διακομιστές του Riseup δεν είναι ο κύριος προορισμός των μηνυμάτων που αποστέλλονται σε 3rd party ψευδώνυμα.

h4. Στέλνοντας email με ένα 3rd party ψευδώνυμο

Εάν προσθέσετε ένα ψευδώνυμο για ένα domain διαφορετικό από του riseup.net, το riseup θα σας επιτρέψει να στέλνετε από τον email client σας χρησιμοποιώντας αυτήν τη διεύθυνση.

Ορισμένες σύγχρονες τεχνολογίες αποκλεισμού ανεπιθύμητων μηνυμάτων όπως [[SPF -> https://en.wikipedia.org/wiki/Sender_Policy_Framework]] και [[DKIM -> https://en.wikipedia.org/wiki/DomainKeys_Identified_Mail]] χρησιμοποιούνται συχνά για τον έλεγχο των διακομιστών που επιτρέπεται επίσημα να στέλνουν μηνύματα από ένα domain.

Εάν το ψευδώνυμό σας βρίσκεται σε domain που έχει διαμορφώσει αυτές τις παραμέτρους τότε οι περισσότεροι διακομιστές στο διαδίκτυο θα αρνηθούν την αλληλογραφία παρόλο που το Riseup σάς επιτρέπει να στείλετε email με αυτό.

h2. Προώθηση email

Με την προώθηση email μπορείτε να προωθήσετε αυτόματα όλα τα μηνύματα ηλεκτρονικού ταχυδρομείου που αποστέλλονται στη riseup διεύθυνση email σας σε άλλη διεύθυνση ηλεκτρονικού ταχυδρομείου.

Μπορείτε να ρυθμίσετε την προώθηση email ακολουθώντας αυτά τα βήματα:

# Συνδεθείτε στο https://account.riseup.net
# Πατήστε στο "Mail Settings"
# Εισαγάγετε τη διεύθυνση στην οποία θα θέλατε να προωθηθεί το riseup email σας στο πεδίο που επισημαίνεται "Forwarding address"
# Πατήστε στο πράσινο κουμπί "Save"
