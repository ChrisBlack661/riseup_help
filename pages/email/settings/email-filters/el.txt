h2. Πώς μπορώ να επεξεργαστώ τα φίλτρα email μου;

# Πρώτα θα πρέπει να βεβαιωθείτε ότι οι φάκελοι που θέλετε να φιλτράρετε υπάρχουν ήδη. Μπορείτε να δημιουργήσετε έναν νέο φάκελο στον email client ή στο webmail σας.
# Συνδεθείτε στο https://account.riseup.net με το riseup όνομα χρήστη και τον κωδικό σας.
# Πατήστε στο *Mail Settings* (στην αριστερή πλευρά της οθόνης).
# Πατήστε στην καρτέλα *Filters*
# Από αυτήν την οθόνη, μπορείτε να αλλάξετε τη σειρά των φίλτρων email σας, να επεξεργαστείτε ή να αφαιρέσετε υπάρχοντα φίλτρα ή να προσθέσετε καινούργια. Για να επεξεργαστείτε ένα υπάρχον φίλτρο πατήστε πάνω του και θα μεταφερθείτε σε μια οθόνη όπου μπορείτε να προσθέσετε νέες συνθήκες, να αφαιρέσετε ή να τις επεξεργαστείτε.

h2. Πώς μπορώ να ρυθμίσω ένα φίλτρο για έναν net φάκελο;

# Ακολουθήστε τα παραπάνω βήματα για να δημιουργήσετε ή να επεξεργαστείτε ένα φίλτρο.
# Προσθέστε έναν χαρακτήρα "." μεταξύ του parent και του child φακέλου (πχ ο φάκελος /riseup/Lists θα εισαχθεί ως "riseup.Lists". Σημειώστε ότι οι φάκελοι είναι case-sensitive και κάνουν διάκριση πεζών-κεφαλαίων).
# Αποθηκεύστε το φίλτο.

h2. Πώς μπορώ να αποκλείσω μια διεύθυνση email;

Εάν λαμβάνετε ενοχλητικά ή ανεπιθύμητα μηνύματα ηλεκτρονικού ταχυδρομείου και θέλετε να διασφαλίσετε ότι μια συγκεκριμένη διεύθυνση ηλεκτρονικού ταχυδρομείου διαγράφεται αυτόματα, τότε ακολουθήστε τα παρακάτω βήματα:

# Συνδεθείτε στο https://account.riseup.net με το riseup όνομα χρήστη και τον κωδικό σας.
# Πατήστε στο *Mail Settings* (στην αριστερή πλευρά της οθόνης).
# Πατήστε στην καρτέλα *Filters*
# Πατήστε στο *Add New Filter*
# Ονομάστε το φίλτρο όπως θέλετε, πχ "Move to trash"
# Κάτω από την ενότητα *messages matching* επιλέξτε *From* και *Contains* και μετά πληκτρολογήστε τη διεύθυνση email, πχ. "george_bush@whitehouse.gov"
# Στο πεδίο *θα τοποθετηθεί σε φάκελο* "Trash" (*case sensitive*- μη χρησιμοποιήσετε "trash" ή "TRASH" γιατί δεν θα λειτουργήσει).
# Πατήστε *Save*

Τελειώσατε! Κάθε email από αυτήν τη διεύθυνση θα μετακινείται αυτόματα στον φάκελο απορριμμάτων σας.

Σημείωση: Εάν θέλετε να προσθέσετε περισσότερες διευθύνσεις email που θέλετε να μετακινούνται στον κάδο απορριμμάτων, μπορείτε απλά να πατήσετε στο *add condition* ακολουθώντας τα παραπάνω βήματα 6-8 - δεν χρειάζεται να δημιουργήσετε νέο φίλτρο.

h2. Πως μπορώ να επιτρέψω emails να περνάνε αυτόματα μέσα από το φίλτρο ανεπιθύμητης αλληλογραφίας;

Εάν υπάρχει κάποιο συγκεκριμένο email που ταξινομείται πάντα στην ανεπιθύμητη αλληλογραφία σας και θέλετε να διασφαλίσετε ότι δεν θα συμβεί στο μέλλον, ακολουθήστε τα παρακάτω βήματα:

# Συνδεθείτε στο https://account.riseup.net με το riseup όνομα χρήστη και τον κωδικό σας.
# Πατήστε στο *Mail Settings* (στην αριστερή πλευρά της οθόνης).
# Πατήστε στην καρτέλα *Spam Settings*.
# Μετακινηθείτε προς τα κάτω μέχρι το *Allow List*. Προσθέστε τη διεύθυνση email που θέλετε να επιτρέψετε μέσω του φίλτρου ανεπιθύμητων μηνυμάτων και πατήστε *Save Changes*.

h2. Μπορώ να ρυθμίσω μια αυτόματη απάντηση out-of-office;

Η Riseup υποστηρίζει απαντήσεις out-of-office!

Όσο είναι ενεργοποιημένο, σε όλα τα εισερχόμενα μηνύματα θα αποστέλλονται αυτοματοποιημένες απαντήσεις. Κάθε παραλήπτης θα λαμβάνει αυτοματοποιημένες απαντήσεις το πολύ μία φορά κάθε δεύτερη μέρα για να ελαχιστοποιηθεί η επισκεψιμότητα μέσω email.

Για να ενεργοποιήσετε τις απαντήσεις out-of-office:

# Συνδεθείτε στο https://account.riseup.net με το riseup όνομα χρήστη και τον κωδικό σας.
# Πατήστε στο *Mail Settings* (στην αριστερή πλευρά της οθόνης).
# Πατήστε στην καρτέλα *Vacation*.
# Συντάξτε ένα μήνυμα out-of-office.
# Αλλάξτε την *Vacation automatic response* ρύθμιση στο "On".
# Πατήστε στο πράσινο κουμπί *Save*.
